FAQ

Common Questions

An access code grants entry to the Supersuite builder. Each plan has unique codes. During beta use: SS26 (Free), SSBASIC (Basic), SSPRO (Pro), or SSAGENCY (Agency). SSV26.2+ will support full email/password authentication.
As the project owner, open the Share panel in the builder and click "Create Invite Code." Share that code with your team. Collaborators get 2 hours/day of editing access. Their changes sync to your project in real time using localStorage broadcast. Collaborators cannot export — exports are owner-only.
The project library lets you save, name, and load multiple website projects locally in your browser. Switch between them anytime from the Projects panel in the sidebar. Free users can store 3 projects; Basic gets 10; Pro and Agency get unlimited.
Clicking "It's Go Time 🚀" downloads a complete standalone index.html with all CSS inlined and full mobile responsiveness. Every style, color, font, block, and layout is 100% retained. Open the file in any browser — no server needed.
Add the "Widget / Embed" block from the block library. Paste any HTML embed code — Elfsight widgets, Tidio chat, Cal.com booking, Stripe Buy buttons, YouTube embeds, or any iframe. The widget renders live in the preview and exports perfectly.
Yes. All projects auto-save to your browser's localStorage every 30 seconds. Your project library persists across sessions. Full cloud sync with user accounts is planned for SSV26.2.
